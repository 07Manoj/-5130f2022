This paper talks about how blockchain could help 5g become more secure and efficient solving for security, data interoparibilty and decentralization. 
The research is still in earlier stages and there's a lot of work that needs to be done to roll out blockchain for 5g.
The authors talk about how technologies like cloud computing, edge computing help 5g be efficient. 
They find that blockchain can support 5G technologies in the following three areas. First, security. Second, system performance and third, resource management
